# Key Bindings

## Global

| Key | Action |
|-----|--------|
| `q` | Quit |
| `?` | Show help overlay |
| `r` / `Ctrl+R` | Reload all files from disk |

## Navigation

| Key | Action |
|-----|--------|
| `↑` / `↓` | Move selection up/down |
| `Tab` | Switch focus: file list ↔ content panel |
| `Enter` | Select highlighted file |
| `Page Up` / `Page Down` | Scroll content panel |

## Viewing

| Key | Action |
|-----|--------|
| `m` | Toggle secret masking on/off |
| `/` or `f` | Open search/filter bar |
| `Esc` | Clear search, close overlays |
| `c` | Copy selected value to clipboard |

## Diff

| Key | Action |
|-----|--------|
| `d` | Toggle diff view |

## File

| Key | Action |
|-----|--------|
| `e` | Open current file in `$EDITOR` (or `notepad` on Windows) |

## Customisation

Key bindings are not yet configurable via a config file — this is on the [roadmap](../roadmap.md). Vote on [this issue](https://github.com/lazyenv/lazyenv/issues) to prioritise it.
